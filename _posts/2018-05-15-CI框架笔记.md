$this->db->last_query();
~~~
2. CI框架中的limit和mysql里面的limit用法相反
&emsp;MySQL里面表示跳过2行取3行
~~~mysql
select * from user limit(2,3);
~~~
&emsp;在CI里面要写成
~~~mysql
select * from user limit(3,2);
~~~
&emsp;即
~~~php
$this->db->limit(4,3);
~~~
